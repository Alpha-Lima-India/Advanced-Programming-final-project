airplane ticket booking

this program is for booking airplane ticket and tasks related to this subject.

Some facts about how program works :
	
	username of any user is unique and we dont have two users with one same usename but passwords are not like this.
	flight numbers are also unique and there can not be two flights with same flight numbers. the date, origin and destination, flight hour and any other informations of a flight are not unique 
	except for the flight number. (in reality we have this rule)
	
	all commands that any user can work with have been written in the program which any user can see them by entering the command "help"
	
	any user in any command must enter the input of the command in the form that command shows at first
